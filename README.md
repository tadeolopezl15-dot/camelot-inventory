# Camelot Inventory Management

A professional English inventory control web app for two warehouses.

## Features

- Dashboard analytics
- Products management
- Warehouse 1 and Warehouse 2 stock tracking
- Stock In
- Stock Out
- Transfers between warehouses
- Low stock alerts
- Recent movement history
- PDF export
- Excel export
- Responsive design for mobile and desktop

## Open in VS Code

1. Unzip the file.
2. Open the folder in VS Code.
3. Open the terminal.
4. Run:

```bash
npm install
npm run dev
```

5. Open the local URL shown in the terminal.

## Online database

This front-end is ready to connect to Supabase. Recommended tables:

- products
- warehouses
- stock_movements
- suppliers
- customers
- users
